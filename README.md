Prism Silex Skeleton
====================

A [Prism][1] [Silex][2] skeleton.

Prism is a URL Shortener and Custom Header Injector.


Requirements
------------

 * PHP 5.3+


Installation
------------

Through [Composer][3] as [offloc/prism-silex-skeleton][4].
